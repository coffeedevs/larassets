# Larassets
[Visit the website](http://larassets.coffeedevs.com/)
Larassets is an online tool for transforming your HTML ``` <script>``` &  ```<link>``` tags, modifying it's ```href``` and ```src``` attributes to use Laravel ```asset()``` helper function.
